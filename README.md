# Evaluación Práctica 1. Resolución de ejercicios


## Descripción
Para nuestra primera evaluación vamos a realizar los siguientes ejercicios, en los cuales deberás aplicar los contenidos desarrollados hasta ahora en la Unidad Curricular. El objetivo es demostrar el dominio de Node.js, TypeScript y controladores.

### Ejercicio 1
Solicitar el nombre de un animal que sea su favorito y, mediante NODE, interceptar y recargar otra página HTML que muestre dicho nombre.
*   **Requisito:** Requiere el uso de dos vistas (una para solicitar el dato y otra para mostrar el resultado).

### Ejercicio 2
Utilizando TypeScript, enumerar:
*   Género de películas.
*   País de la película.
Luego, mostrarlos en pantalla.

## Recomendaciones
*   Todo el código fuente debe ser documentado con las palabras del estudiante (explicando la lógica propia), no con descripciones genéricas o generadas automáticamente por el framework.
*   La entrega de los resultados debe realizarse siguiendo las indicaciones dadas por el docente.
